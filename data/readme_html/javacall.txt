<p>Call Java programs from Julia.</p>
<p>Documentation is available at <a href="http://juliainterop.github.io/JavaCall.jl" rel="nofollow">http://juliainterop.github.io/JavaCall.jl</a></p>
<p><em>JavaCall and it's derivatives do not work correctly Julia 1.1 and Julia 1.2. On Julia 1.3, please set the environment variable <code>JULIA_COPY_STACKS</code>. On 1.1 and 1.2, and on 1.3 without <code>JULIA_COPY_STACKS</code> set, you may see segfaults or incorrect results. This is typically due to stack corruption. The Julia long-term-support version of 1.0.x continues to work correctly as before.</em></p>
