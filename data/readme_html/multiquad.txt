<p><strong>MultiQuad.jl</strong> is a convenient interface to perform 1D, 2D and 3D numerical integrations.
It uses <a href="https://github.com/JuliaMath/QuadGK.jl">QuadGK</a>, <a href="https://github.com/giordano/Cuba.jl">Cuba</a> and <a href="https://github.com/stevengj/HCubature.jl">HCubature</a> as back-ends.</p>

<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/1ca71382ce9b941233f43381a2a3f3f01fdc9d51/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f5c696e745f7b78317d5e7b78327d662878296478"><img src="https://camo.githubusercontent.com/1ca71382ce9b941233f43381a2a3f3f01fdc9d51/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f5c696e745f7b78317d5e7b78327d662878296478" title="\int_{x1}^{x2}f(x)dx" data-canonical-src="https://latex.codecogs.com/gif.latex?\int_{x1}^{x2}f(x)dx" style="max-width:100%;"></a></p>

<p>It is suggested to use <code>:quadgk</code> (the default option).</p>
