<p><code>Coluna</code> is a branch-and-price-and-cut framework that decomposes and solves
a mixed-integer program (MIP). The user introduces his "original" problem formulation using the <a href="https://github.com/JuliaOpt/JuMP.jl"><code>JuMP</code></a> modeling language and our specific extension
<a href="https://github.com/atoptima/BlockDecomposition.jl"><code>BlockDecomposition</code></a>.</p>
<p><code>Coluna</code> reformulates the user problem based on the "annotations" provided along side the original formulation using <code>BlockDecomposition</code>.</p>
<p><code>Coluna</code> aims to be very modular and tweakable so that any user can define the behavior of his customized branch-and-price-and-cut algorithm. The user can create its own algorithmic strategy using the algorithmic building blocks offered in <code>Coluna</code>.</p>

branch-and-cut-and-price algorithms.</p>
<p>As functionality goes, we aim to provide the support for:</p>
