ORCA.jl Julia interface to plotly's ORCA tool for generating static files based on plotly graph specs. Usage There is a single exported function savefig . After loading PlotlyBase.jl or PlotlyJS.jl and creating a plot named p you can call savefig(p::Union{Plot,SyncPlot}, filename::String; options...) to save the plot in p to a file named filename . See docstring for a description of options .
