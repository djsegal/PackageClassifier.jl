DocStringExtensions Extensions for Julia's docsystem. Documentation Build Status Installation The package can be added using the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run pkg> add DocStringExtensions Documentation STABLE — most recently tagged version of the documentation. LATEST — in-development version of the documentation. Project Status The package is tested and developed against Julia 0.7 and 1.0 on Linux, OS X, and Windows, but there are versions of the package that works on older versions of Julia.
