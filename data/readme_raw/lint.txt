Lint.jl Lint.jl is a tool that uses static analysis to assist in the development process by detecting common bugs and potential issues. Installation Lint.jl can be installed through the Julia package manager: Pkg . add ( " Lint " ) Usage There are 3 functions you can use to lint your code. lintpkg("MyPackage") for linting an entire package lintfile("my_file.jl") for linting a file lintstr("my string") for linting a string Documentation Detailed documentation is available for: Latest Release Development
