CountdownLetters This package solves letters puzzles as seen on the television show Countdown. Installation julia > import Pkg; Pkg . add ( " CountdownLetters " ) Examples You can provide the input letters as either a single string or a vector of characters. julia > using CountdownLetters julia > letters ( " GYHDNOEUR " ) 270 - element Array{String, 1 } : " greyhound " " hydrogen " ⋮ julia > letters ([ ' G ' , ' Y ' , ' H ' , ' D ' , ' N ' , ' O ' , ' E ' , ' U ' , ' R ' ]) 270 - element Array{String, 1 } : " greyhound " " hydrogen " ⋮
