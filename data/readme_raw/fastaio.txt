FastaIO.jl Documentation Build Status Utilities to read/write FASTA format files in Julia . Installation and usage Installation To install the module, use Julia's package manager: start pkg mode by pressing ] and then enter: (v1.3) pkg> add FastaIO Dependencies will be installed automatically. The module can then be loaded like any other Julia module: julia> using FastaIO Documentation STABLE — most recently tagged version of the documentation. DEV — in-development version of the documentation. See also the examples in the examples/ directory.
