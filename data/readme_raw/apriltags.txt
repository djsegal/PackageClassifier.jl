AprilTags This package is a ccall wrapper for the AprilTags library tailored for Julia. Installation AprilTags.jl can be installed in Julia 0.7 and 1.0 with: # enter ']' to get the package manager and then type: (v0 .7 ) pkg > add AprilTags # or using Pkg Pkg . add ( " AprilTags " ) Please see v0.0.2 for julia 0.6 support. See documentation , examples, and test folder for usage. Examples To run the examples cd to the project directory and call: (v1.0) pkg> activate . (AprilTags Examples) pkg> instantiate
