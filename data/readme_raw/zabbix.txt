Installation Since Zabbix is registered in METADATA.jl , you can directly install it like, julia > Pkg . add ( " Zabbix " ) Update julia > Pkg . update ( " Zabbix " ) How it works? Please make sure that you have a valid Zabbix URL available along with the right access levels. In case you have only read access on the zabbix server, you may not be able to use configuration and management operations. Full read & write access is desired.
