OpenEphysLoader.jl A set of tools to load data written by the Open Ephys GUI Documentation Build Status Requirements Julia 0.6 or higher Installation This package is registered, so simply install it with the package manager: julia > Pkg . add ( " OpenEphysLoader " ) Documentation STABLE — most recently tagged version of the documentation. LATEST — in-development version of the documentation. Project Status This package is tested against Julia 0.6 , 0.7 , 1.0 , 1.1 , and nightlies on Linux, OS X, and Windows. This package only supports reading from continuous files at the moment, with no immediate plans to support spike data. Contributing Contributions are welcome, as are feature requests and suggestions. Please open an issue if you encounter any problems.
