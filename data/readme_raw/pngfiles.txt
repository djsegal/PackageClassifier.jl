PNGFiles.jl (work in progress) FileIO.jl integration for PNG files Installation Install with Pkg, just like any other registered Julia package: pkg > add PNGFiles # Press ']' to enter te Pkg REPL mode. Usage PNGFiles is not yet integrated into FileIO. For now, you can load png files using: using PNGFiles PNGFiles . save ( " path/to/img.png " , rand (Gray, 100 , 100 )) PNGFiles . load ( " path/to/img.png " ) In the future, it will be as simple as: using FileIO save ( " test.png " , rand (Gray, 100 , 100 )) load ( " test.png " )
