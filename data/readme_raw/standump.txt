StanDump Julia package for saving data (arrays and scalars) in a format that can be read by cmdstan . Example usage N = 200 stan_dump ( " /tmp/test.data.R " , (N = N, x = randn (N))) stan_dump(target, data) is the main entry point. The first argument is usually a filename (see its docstring for other options), while the data is specified as a NamedTuple . Variable names are minimally validated. Only supports types understood by cmdstan .
