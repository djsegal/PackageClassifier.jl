LeapSeconds Leap seconds in Julia A new minor version of this package will be published everytime a new leap second is issued be the IERS and dependent packages will need to be updated! Installation pkg > add LeapSeconds Usage The package exports a single function offset_tai_utc which returns the offset between International Atomic Time (TAI) and Coordinated Universal Time (UTC) for a given date. For dates after 1972-01-01, this is the number of leap seconds. using LeapSeconds using Dates dt = DateTime ( 2017 , 1 , 1 ) # Pass a `DateTime` instance... offset_tai_utc (dt) # ...or a Julian Date offset_tai_utc ( datetime2julian (dt))
