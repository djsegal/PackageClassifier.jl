CrossfilterCharts.jl A Julia package for automagical DC.js linked charts in your IJulia notebook - used to be DC.jl. Demonstration IJulia Notebook . Install Quickly take a data set and make exploratory data visualizations with just a few lines of code. In a Julia terminal run: Pkg . add ( " CrossfilterCharts " ) Docs Documentation is available both via the Documenter.jl badges at the top of this page or in a demonstration IJulia notebook .
