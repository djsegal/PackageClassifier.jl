PValueAdjust.jl This package has been deprecated in favor of MultipleTesting.jl . Please use that package instead. Some methods to adjust p-values for multiple comparisons in Julia. The various methods can be called using the function padjust . padjust takes an array of p-values and a second method parameter and returns an array of adjusted p-values. Please refer to the documentation of the corresponding function in R or to Wikipedia ( FWER , FDR ), if you want to know more on this topic. Current stable version is 2.0.0 .
