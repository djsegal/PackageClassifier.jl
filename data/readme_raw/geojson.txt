GeoJSON.jl This library is developed independently of, but is heavily influenced in design by the python-geojson package. It contains: Functions for encoding and decoding GeoJSON formatted data a type hierarchy (according to the GeoJSON specification ) An implementation of the _ geo_interface_ , a GeoJSON-like protocol for geo-spatial (GIS) vector data. Documentation Documentation for GeoJSON.jl can be found at https://juliageo.github.io/GeoJSON.jl/dev/ .
