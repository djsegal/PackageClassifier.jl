FlameGraphs FlameGraphs is a package that adds functionality to Julia's Profile standard library. It is directed primarily at the algorithmic side of producing flame graphs , but includes some "format agnostic" rendering code. You might use FlameGraphs on its own, but users should consider one of its downstream packages: ProfileView , a graphical user interface (GUI) based on Gtk for displaying and interacting with flame graphs Juno , a development environment that supports profile visualization ProfileVega , the recommended package for display in Jupyter notebooks ProfileSVG , a package for writing flame graphs to SVG format See the documentation for details.
