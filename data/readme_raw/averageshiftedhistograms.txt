Docs Build Test AverageShiftedHistograms Lightning fast density estimation in Julia. An Averaged Shifted Histogram (ASH) is essentially Kernel Density Estimation over a fine-partition histogram. ASH only requires constant memory and can be constructed on-line, allowing you to estimate distributions for arbitrarily big data.
