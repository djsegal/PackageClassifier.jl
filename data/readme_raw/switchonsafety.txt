Switch On Safety (SOS) Documentation Build Status References to cite This packages implements methods for computing invariant sets using Sum Of Squares Programming . It supports: Systems defined in MathematicalSystems.jl . Hybrid Systems defined in HybridSystems.jl . It also includes utilities for approximation the Joint Spectral Radius . Installation The package currently requires Julia v1.0, you can download it here . Once Julia is installed, simply launch the REPL an type ] add SwitchOnSafety Examples Example notebooks are available in the examples folder . We link them below with the literature.
