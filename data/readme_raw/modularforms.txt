ModularForms.jl implementation of modular forms algorithms in Julia Getting started Install Julia . Install Nemo : at the Julia prompt, press ] to access the package manager, then: (v1.0) pkg> add Nemo Press Backspace to exit the package manager. Start Julia from ModularForms.jl directory.
