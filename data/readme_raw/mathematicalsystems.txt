MathematicalSystems.jl MathematicalSystems is a Julia package for mathematical systems interfaces. Resources Manual Contributing Release notes Release notes of the development version Installing This package requires Julia v1.0 or later. Refer to the official documentation on how to install and run Julia in your system.
