Merlin: deep learning framework for Julia Merlin is a deep learning framework written in Julia . It aims to provide a fast, flexible and compact deep learning library for machine learning. Merlin is tested against Julia 1.0 on Linux, OS X, and Windows (x64). It runs on CPU and CUDA GPU.
