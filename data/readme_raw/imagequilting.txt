ImageQuilting.jl A Julia package for fast 3D image quilting simulation. Latest News ImageQuilting.jl won the Syvitski Modeler Award 2018 . ImageQuilting.jl can now be used as one of the many solvers in the GeoStats.jl framework. For more information, please type ?ImgQuilt in the Julia prompt after loading the package.
