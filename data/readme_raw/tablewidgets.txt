TableWidgets This package contains a few basic widgets to build GUIs to work with tabular data as well as examples of such GUIs. Examples To run the examples, you need to activate the respective folder and instantiate it to get all dependencies. import Pkg import TableWidgets : examplefolder Pkg . activate (examplefolder) Pkg . instantiate () include ( joinpath (examplefolder, " explorer " , " sputnik.jl " ))
