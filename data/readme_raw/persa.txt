Persa.jl Collaborative Filtering in Julia Documentation Build Status Installation The package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run: pkg> add Persa Or, equivalently, via the Pkg API: julia > import Pkg; Pkg . add ( " Persa " ) Packages Some important packages: DatasetsCF : Package with collaborative filtering datasets. EvaluationCF : Packages for evaluation of predictive algorithms. It contains metrics, data partitioning and more. (under construction) ModelBasedCF : Model based algorithms. (under construction)
