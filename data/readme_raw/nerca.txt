NeRCA NeRCA ( Ne utrino R esearch with C osmics in the A byss) is a Julia package to access and analyse KM3NeT related data. It can read the official KM3NeT HDF5 formats, calibration information (DETX files) and also live data from running detectors. The ROyFit reconstruction is also included and can be found in src/fit.jl .
