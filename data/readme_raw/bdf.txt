BDF.jl is a Julia module to read/write BIOSEMI 24-bit BDF files (used for storing electroencephalographic recordings) Usage: bdfHeader = readBDFHeader("res1.bdf") #read the bdf header sampRate = bdfHeader["sampRate"][1] #get the sampling rate #read the data, the event table, the trigger channel and the status channel dats, evtTab, trigs, statusChan = readBDF("res1.bdf") Documentation is available here: http://samcarcagno.altervista.org/BDF/index.html
