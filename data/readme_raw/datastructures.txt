DataStructures.jl This package implements a variety of data structures, including Deque (implemented with an unrolled linked list ) CircularBuffer CircularDeque (based on a circular buffer) Stack Queue Priority Queue Fenwick Tree Accumulators and Counters (i.e. Multisets / Bags) Disjoint Sets Binary Heap Mutable Binary Heap Ordered Dicts and Sets RobinDict (implemented with Robin Hood Hashing ) Dictionaries with Defaults Trie Linked List and Mutable Linked List Sorted Dict, Sorted Multi-Dict and Sorted Set DataStructures.IntSet SparseIntSet Resources Documentation : https://juliacollections.github.io/DataStructures.jl/latest
