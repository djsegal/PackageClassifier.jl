DistributedArrays Distributed arrays for Julia. Documentation Build Status Introduction DistributedArrays.jl uses the stdlib Distributed to implement a Global Array interface. A DArray is distributed across a set of workers. Each worker can read and write from its local portion of the array and each worker has read-only access to the portions of the array held by other workers. Installation The package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run: pkg> add DistributedArrays Or, equivalently, via the Pkg API: julia > import Pkg; Pkg . add ( " DistributedArrays " ) Documentation STABLE — documentation of the most recently tagged version.
