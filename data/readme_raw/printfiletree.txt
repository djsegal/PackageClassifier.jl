PrintFileTree Exports a single utility function, printfiletree(path) . Compatability Julia v0.6 , v0.7 , v1.0 , v1.1 + printfiletree(path) Prints a file tree rooted at path, in the same way as the Unix utility, tree . Example: julia > printfiletree ( " my/files " ) my / files ├── a . txt ├── b . png ├── c │ ├── a │ │ ├── a │ │ │ └── subfile │ │ └── subfiles │ ├── cats │ │ └── are │ │ └── so │ │ └── cool │ └── cool └── d 8 directories, 5 files Installation julia v0.6: julia> Pkg.add("PrintFileTree") julia v0.7+: pkg> add "PrintFileTree" Based on http://mama.indstate.edu/users/ice/tree/
