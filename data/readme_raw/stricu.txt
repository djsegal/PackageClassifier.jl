StrICU: International Components for Unicode (ICU) wrapper for Julia ==================================================================== Julia wrapper for the International Components for Unicode (ICU) libraries . Info Windows Linux & MacOS Package Evaluator CodeCov Coveralls This is a new wrapper for the ICU library, designed to work on Julia v0.6 and above, using the Strs.jl package to provide support for UTF-16 encoded strings. The API has been redesigned to not pollute the namespace and to try to be a bit more "Julian"
