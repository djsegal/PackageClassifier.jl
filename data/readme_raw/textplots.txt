TextPlots.jl Fancy terminal plotting for Julia using Braille characters. Inspired by Drawille but not based on it due to its restrictive license (AGPL). TextPlots.jl is free for all under the MIT license. TextPlots.jl aims for as much elegance and aesthetics as possible for a terminal (unicode) plotting library. It should be able to plot any continuous real-valued function, and any small-to-medium collection of points. Screenshots taken in iTerm2 on a Mac using the Adobe Source Code Pro font . INSTALLATION TextPlots.jl currently has no dependencies on other packages. It has only been tested against Julia v0.3 nightly builds on a Mac. julia > Pkg . add ( " TextPlots " ) julia > using TextPlots USAGE TextPlots.jl is very simple to use: just pass a function and a
