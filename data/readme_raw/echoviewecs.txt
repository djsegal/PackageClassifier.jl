EchoviewEcs.jl Julia package for reading Echoview calibration supplement (.ECS) files . Scientific echosounder data requires calibration correction, and ECS is a popular storage file format for the correction parameters. using EchoviewEcs calibrations = load(filename) calibrations is a Vector of Dict where Dict contains keys and values being configuration parameters. SourceCal settings inherit from FileSet settings but override such settings if specified explicitly.
