GmshTools.jl To use Gmsh mesh program. Notice Users who would like try the newest version could set the environment variable GMSH_LIB_PATH to your source-compiled Gmsh library directory and rebuild this package. This needs only to be done once. To come along nicely with Intel MKL, the Julia must be compiled with 32 integer interface, i.e. linking to lp64 instead of ipl64 .
