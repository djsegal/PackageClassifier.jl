PhysOcean Content You will find here some general tools for Physical Oceanography (state equations, bulk formulas, geostrophic velocity calculations ...). Installing Your need Julia to use PhysOcean . The command line version is sufficient for PhysOcean . Inside Julia, you can download and install the package by issuing: using Pkg Pkg . add ( " PhysOcean " ) Or if you want to use the latest version, you can use the following command: using Pkg Pkg . add ( PackageSpec (name = " PhysOcean " , rev = " master " )) Testing A test script is included to verify the correct functioning of the toolbox. The script should be run in a Julia session. using Pkg Pkg . test ( " PhysOcean " )
