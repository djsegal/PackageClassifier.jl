ProgressLogging: a package for defining progress logs ProgressLogging.jl is a package for defining progress logs . It can be used to report progress of a loop/loops with time-consuming body: julia > using ProgressLogging julia > @progress for i in 1 : 10 sleep ( 0.1 ) end This package does not contain any progress monitors for visualizing the progress of the program. You need to install a package supporting progress logs created by ProgressLogging.jl API. For example: Juno TerminalLoggers.jl
