ImageInTerminal ImageInTerminal.jl is a drop-in package that once imported changes a how a single Colorant and whole Colorant arrays (i.e. Images) are displayed in the interactive REPL. The displayed images will be downscaled to fit into the size of your active terminal session. To activate this package simply import it into your Julia session. By default this packages uses 256 colors. If your terminal supports 24 bit colors you can make use of them by typing the following: using ImageInTerminal ImageInTerminal . use_24bit () Without this package julia > using Images, TestImages If you see out of place horizontal lines in your Image it means that your font displays the utilized unicode block-characters in an unfortunate way. Try changing font or reducing your terminal's line-spacing. If your font is Source Code Pro, update to the latest version.
