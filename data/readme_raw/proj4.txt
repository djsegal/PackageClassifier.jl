Proj4.jl A simple Julia wrapper around the PROJ cartographic projections library. Basic example: using Proj4 wgs84 = Projection ( " +proj=longlat +datum=WGS84 +no_defs " ) utm56 = Projection ( " +proj=utm +zone=56 +south +datum=WGS84 +units=m +no_defs " ) transform (wgs84, utm56, [ 150 - 27 0 ]) # Should result in [202273.913 7010024.033 0.0] API documentation for the underlying C API may be found here: https://proj.org/development/reference/index.html
