LanguageServer Overview This package implements the Microsoft Language Server Protocol for the Julia programming language. Text editors with a client for the Language Server Protocol are able to make use of the Julia Language Server for various code editing features: VS Code Atom Vim and Neovim Emacs Sublime Text Installation and Usage Documentation : using Pkg Pkg . add ( " LanguageServer " ) Instantiate an instance of the language server with LanguageServerInstance and run it: using LanguageServer server = LanguageServerInstance (stdin, stdout, false , " /path/to/environment " ) run (server)
