LinearCovarianceModels LinearCovarianceModels.jl is a package for computing Maximum Likelihood degrees and MLEs of linear covariance models using numerical nonlinear algebra. In particular HomotopyContinuation.jl . Installation In order to use LinearCovarianceModels.jl you need to have at least Julia 1.1 installed. If this is not the case you can download it at julialang.org . Please see the platform specific instructions if you have trouble installing Julia. The package can be installed by executing julia > using Pkg; Pkg . add ( " LinearCovarianceModels " ) in the Julia REPL. If you are looking for a more IDE like experience take a look at Juno .
