PyCallJLD.jl PyCallJLD enables saving and loading PyCall's PyObject s using JLD.jl . Example: using PyCall, JLD, PyCallJLD @pyimport sklearn . linear_model as lm # Create some Python objects m1 = lm . LinearRegression () m2 = lm . ARDRegression () # Save them to models.jld JLD . save ( " models.jld " , " mods " , [m1, m2]) # Load them back models = JLD . load ( " models.jld " , " mods " ) The objects are serialized using cPickle.dumps See PyCall 's and JLD 's documentation for details.
