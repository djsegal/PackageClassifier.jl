AgentsPlots.jl Plotting functionality for Agents.jl agent-based modeling in Julia. Agents.jl is a Julia framework for agent-based modeling (ABM). It provides a structure and components for quickly implementing agent-based models, run them in batch, collect data, and visualize them. AgentsPlots.jl is plotting functionality for 2D grid models in Agents.jl Installation Install using the following command inside Julia: ]add AgentsPlots It is compatible with Julia 1.0+.
