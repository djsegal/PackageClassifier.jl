Ditherings Dithering Algorithms for Julia Basic Usage Examples julia> using Ditherings Simple Reduced Precision Colour julia> img = load("lenna.png"); Switch to zero or one per pixel julia> Ditherings.FloydSteinbergDither4Sample(img, Ditherings.ZeroOne) Switch to zero or one per channel julia> Ditherings.FloydSteinbergDither4Sample(img, Ditherings.ZeroOne_PerChannel) Approach The aim of these funcions is that you can call a function which represents an error diffusion shape, and then pass in a palette function which maps the input function into the reduced precision space. Optionally you can also pass in custom weights for the error diffusion kernel
