SchwarzChristoffel Documentation Build Status About the package The purpose of this package is to enable easy construction and evaluation of the mapping from the region inside or outside the unit circle to the exterior of a closed polygon. The engine for constructing the mapping and its inverse is based on the work of Driscoll and Trefethen, Schwarz-Christoffel Mapping , Cambridge University Press, 2002. Installation This package requires Julia 1.0 and above. It is a registered package, so it should be installed with add SchwarzChristoffel at the package manager prompt. Examples can be found in the documentation .
