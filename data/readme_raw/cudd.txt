CUDD.jl A Julia wrapper for the CUDD C library to manipulate Algebraic Decision Diagrams (ADDs). Basic usage: >>> using CUDD >>> manager = initialize_cudd () >>> x1 = add_var (manager) >>> ref (x1) >>> x2 = add_var (manager) >>> ref (x2) >>> f = add_apply (manager, add_plus_c, x1, x2) >>> get_value ( evaluate (manager, f, [ 1 , 1 ])) 2 For further examples, see the Julia notebook in the docs folder.
