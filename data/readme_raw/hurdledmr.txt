HurdleDMR Documentation Linux/MacOS/Windows Code HurdleDMR.jl is a Julia implementation of the Hurdle Distributed Multinomial Regression (HDMR), as described in: Kelly, Bryan, Asaf Manela, and Alan Moreira (2018). Text Selection. Working paper . It includes a Julia implementation of the Distributed Multinomial Regression (DMR) model of Taddy (2015) . Read the documentation and tutorials to learn more.
