NLPModelsJuMP Cite as Abel Soares Siqueira & Dominique Orban. NLPModelsJuMP.jl. Zenodo. http://doi.org/10.5281/zenodo.2574164 This package provides nonlinear programming models as implemented by NLPModels , through the use of JuMP . Disclaimer : NLPModelsJuMP is not developed or maintained by the JuMP developers. See the documentation on NLPModels for the general description of NLPModels. Here, we focus on the use of JuMP to create these. NLPModels from a JuMP NLPModelsJuMP provides conversion between MathProgBase models and NLPModels.
