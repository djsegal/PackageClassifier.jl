The aim of this package is to give access to common geographics datasets. Currently it contains: the global land-sea-river mask from GMT the Global Self-consistent, Hierarchical, High-resolution Geography Database (gshhg) . Contributions are most welcome. Here are some guidelines: data sets should be global in scope available under a license allowing redistribution hosted on a publicly available server Installation Install Julia 1.0 (or later) and issue the commands: using Pkg Pkg . add ( " GeoDatasets " ) Documentation Documentation is available here .
