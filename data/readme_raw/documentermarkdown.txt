DocumenterMarkdown Build Status This package enables the Markdown / MkDocs backend of Documenter.jl . Installation The package can be added using the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run pkg> add DocumenterMarkdown Usage To enable the backend import the package in make.jl and then just pass format = Markdown() to makedocs : using Documenter using DocumenterMarkdown makedocs (format = Markdown (), ... )
