Singular Julia package for using the Singular library for commutative and non-commutative algebra, algebraic geometry, and singularity theory. To build Singular.jl, start julia and then type: julia > using Pkg julia > Pkg . clone ( " https://github.com/oscar-system/Singular.jl " ) julia > Pkg . build ( " Singular " ) To use Singular.jl, start julia and then type: julia > using Singular Documentation is available here: https://oscar-system.github.io/Singular.jl/latest/index.html
