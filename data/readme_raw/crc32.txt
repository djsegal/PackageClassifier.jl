Overview See http://en.wikipedia.org/wiki/Cyclic_redundancy_check for more information. This package is deprecated. Please use the CRC32 implementation in any of the following libraries: Zlib.jl -- depends on zlib but about 26x faster CRC.jl -- written in pure Julia and is comparable in speed to Zlib.jl Installation Install via the Julia package manager, Pkg.add("CRC32") . Usage julia > using CRC32 julia > crc = crc32 ( " hello " ) 0x3610a686 julia > crc32 ( " , world " , crc) 0xffab723a julia > crc32 ( " hello, world " ) 0xffab723a
