FStrings.jl Loose implementation of Python style fstring literal string interpolation based on Printf.@sprintf . Usage Examples julia > using FStrings julia > f " π = {π:.2f} " " π = 3.14 " julia > x = 30 julia > f " 0x{x+1:02x} " " 0x1f " Format Specifiers Please refer to Printf.@sprintf for further details on the available format specifiers. Also refer to the principle syntax of fstring literal string interpolation .
