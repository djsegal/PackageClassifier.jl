DocumenterLaTeX Build Status This package enables the LaTeX / PDF backend of Documenter.jl . Installation The package can be added using the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run pkg> add DocumenterLaTeX Usage To enable the backend import the package in make.jl and then just pass format = DocumenterLaTeX.LaTeX() to makedocs : using Documenter using DocumenterLaTeX makedocs (format = DocumenterLaTeX . LaTeX (), ... )
