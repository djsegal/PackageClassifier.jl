BIAS Bayesian Inference At Scale BIAS is an open source implementation of several Bayesian parametric and non-parametric models in julia . The implemented models include: Bayesian Mixture Model (BMM) Latent Dirichlet Allocation (LDA) Dirichlet Process Mixture Model (DPM) Hierarchical Dielectric Mixture Model (HDP) dynamic Hierarchical Dirichlet Mixture Model (dHDP) Temporal Dirichlet Process Mixture model (TDPM) also called Recurrent Chinese Restaurant Process (RCRP) Recurrent Chinese Restaurant Franchise (RCRF) See the Package Documentation for details and examples.
