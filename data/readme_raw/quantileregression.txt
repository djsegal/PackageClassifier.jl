Quantile regression in the Julia language A very simple (and mostly untested) implementation of quantile regression. Install using Pkg.clone("https://github.com/pkofod/QuantileRegression.jl") Main author: Patrick Kofod Mogensen Contact: Use the https://github.com/pkofod/QuantileRegression.jl/issues page License: BSD-3 Example The file examples/qreg_example.jl shows how to use the functions provided here. It replicates part of the analysis in: Koenker, Roger and Kevin F. Hallock. "Quantile Regression".
