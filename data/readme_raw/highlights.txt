Highlights A source code highlighter for Julia. Documentation Build Status Installation The package is registered and can be installed using the package manager. In the Julia REPL type ] to enter the package manager REPL mode and run pkg> add Highlights Documentation STABLE — most recently tagged version of the documentation. LATEST — in-development version of the documentation. Project Status The package is tested against the released Julia 1.x versions and the nightly builds on Linux, OS X, and Windows.
