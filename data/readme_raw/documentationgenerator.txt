DocumentationGenerator.jl Generate documentation for all the packages . Installation pkg> add DocumentationGenerator installs this package. Optionally you might want to install the commonmarker and licensee ruby gems for better license detection and GFM rendering: $ gem install commonmarker $ gem install licensee Usage Running ./run/makedocs.jl will generate the documentation for all packages installable on the current Julia version in ./build . ./dist will contain the correct directory structure. Executing ./run/generate_json.jl creates a packages_meta.json in ./build , which powers the frontend in ./landingpage .
