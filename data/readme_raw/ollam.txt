OLLAM: Online Learning of Linear Adaptatable Models This package is an implementation of: MIRA and PA algorithms (Krammer and Singer 2003) Averaged Perceptron (Collins 2001) It is also contains a LIBSVM wrapper for linear models (mainly for comparison and initialization). Prerequistes Stage.jl - Needed for logging and memoization (Note: requires manual install) LIBSVM.jl - LibSVM binaries and julia wrapper MNIST.jl - for testing Install This is an experimental package which is not currently registered in the julia central repository. You can install via: Pkg . clone ( " https://github.com/saltpork/Stage.jl " ) Pkg . clone ( " https://github.com/mit-nlp/Ollam.jl " ) This process should install all dependent packages in addition to Ollam .
